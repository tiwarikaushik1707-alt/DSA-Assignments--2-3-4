# Sorting Performance Analyzer (SPA)

A Python-based project that implements and compares the performance of three sorting algorithms: Insertion Sort, Merge Sort, and Quick Sort.

## Features

* Implements sorting algorithms from scratch
* Generates datasets (random, sorted, reverse)
* Measures execution time for fair comparison
* Outputs results in a structured format

## How to Run

```bash
python spa_sorting.py
```

## Output

* `output.txt` → Contains correctness check and timing results
* `report.pdf` → Analysis and comparison of algorithms

## Algorithms Used

* Insertion Sort → O(n²), best for small or nearly sorted data
* Merge Sort → O(n log n), stable and consistent
* Quick Sort → O(n log n) average, fastest in practice

## Notes

* No built-in sorting functions are used
* Same timing method applied for all algorithms
