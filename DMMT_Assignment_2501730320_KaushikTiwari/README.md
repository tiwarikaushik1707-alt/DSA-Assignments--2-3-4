# 📊 Data Management Mini Toolkit (DMMT)

A Python project implementing key data structures: **BST, Graph, and Hash Table**.

---

## 🚀 Features

* 🌳 **BST** → Insert, Search, Delete (all cases), Inorder traversal
* 🌐 **Graph** → Adjacency list, BFS & DFS
* 🧩 **Hash Table** → Separate chaining with collision handling

---

## 📂 Structure

```
dmmt_toolkit.py
output.txt
report.pdf
```

---

## ▶️ Run

```
python dmmt_toolkit.py
```

---

## 🎯 Purpose

Understand how data is stored, searched, and traversed efficiently in real systems like databases, networks, and caching.
